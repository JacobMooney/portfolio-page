CanWeListen?

CanWeListen is an app aimed at providing better information for impaired movie-goers. Users search for theaters in their area, view accessability options, and post their own ratings about their experiences.

A site with a goal of demystifying fitness and getting people into the gym. The project includes information about nutrition, fitness, with interactive components and a routine builder.

A small message board where visitors can leave a message with an image or gif. The front end was created using React and Bootstrap, while the backend uses Express and a small middleware called Monk to connect to MongoDB Atlas.

A small project to practice API use, but also because these movies are fantastic. Accesses and displays the movies produced by Studio Ghibli, along with some information about each film.
